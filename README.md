# Student Expense Analyzer

A data analytics web app built with Python and Streamlit that helps students track their monthly expenses, visualize spending patterns, and compare their habits against real student data from Kaggle.

## Features
- Track expenses by category (Food, Transport, Supplies, etc.)
- Real-time budget tracking with over-budget warnings
- Spending breakdown with percentages
- Compare your spending to average students using a Kaggle dataset
- Personalized improvement suggestions

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Kaggle Student Spending Dataset
